# Cyber Report Format Reference

## Markdown Format

Reports are generated by `CyberReportGenerator.generate()` which returns `(markdown: str, json: dict)`.

### Mandatory Sections (11)

```markdown
# Cyber Security Report — Mission `{mission_id}`
*Generated: YYYY-MM-DD HH:MM UTC*

## 1. Executive Summary
{N} finding(s): {confirmed} confirmed, {candidate} candidate, {unverified} unverified.
[> ⚠️ APPROVAL REQUIRED (if approval_required=True)]

## 2. Scope
{scope_summary}

## 3. Authorization
Authorization status: **{authorization_status}**

## 4. Methodology
Static analysis only — no live exploitation, no brute force, no post-exploitation.
Actions performed: [list]

## 5. Findings
### Finding N: {title} — {STATUS_MARK}
- **Class**: `{vuln_class}`
- **Severity**: `{severity}`
- **Confidence**: {confidence%}
- **Location**: `file::function`

**Description**: {description}
**Impact**: {impact}
**Remediation**: {remediation}
[> ⚠️ UNVERIFIED notice if applicable]

## 6. Evidence Summary
{evidence_summary}

## 7. Impact
{risk_summary}

## 8. Recommended Fixes
{remediation_plan}

## 9. Tests to Add
- {regression_tests_recommended}

## 10. Limitations
- {limitations}

## 11. Blocked Actions
- `exploit`, `brute_force`, ... (transparency section)
```

### Status Marks

| Status | Mark |
|--------|------|
| CONFIRMED | ✅ CONFIRMED |
| UNVERIFIED | ⚠️ UNVERIFIED |
| CANDIDATE | 🔍 CANDIDATE |
| FALSE_POSITIVE | ❌ FALSE POSITIVE |
| FIXED | 🟢 FIXED |

## Redaction Rules

All string fields pass through `core.observability.redactor.redact()`:

| Pattern | Replacement |
|---------|-------------|
| `sk-[A-Za-z0-9]{20+}` | `[API_KEY_REDACTED]` |
| `Bearer <token>` | `Bearer [TOKEN_REDACTED]` |
| `bea-<token>` | `[BEA_TOKEN_REDACTED]` |
| `email@domain` | `[EMAIL_REDACTED]` |
| Long opaque strings (40+ chars) | `[LONG_STRING_REDACTED]` |

## Example

```python
from agent_cyber.reports import CyberReport, CyberReportGenerator

report = CyberReport(
    mission_id="m-001",
    scope_summary="Local codebase review of /app/",
    authorization_status="explicit",
    findings=[...],
)

gen = CyberReportGenerator()
markdown, json_data = gen.generate(report)
```
