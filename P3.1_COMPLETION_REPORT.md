## Scoring System

**Weights (total 100%):**
- **Demand:** 35% (upvotes, comments, mentions)
- **Competition:** 20% (existing solutions, market saturation)
- **Feasibility:** 25% (tech complexity, time to build)
- **Monetization:** 20% (willingness to pay, pricing potential)

**Formula:**
```python
total_score = (
    demand_score * 0.35 +
    competition_score * 0.20 +
    feasibility_score * 0.25 +
    monetization_score * 0.20
)
```

**Thresholds:**
- **Excellent:** ≥ 85 (Telegram alert)
- **High:** ≥ 70 (Store in DB)
- **Medium:** ≥ 50 (Log only)
- **Low:** < 50 (Discard)
